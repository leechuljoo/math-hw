적분인자(Integrating Factor) 공식 증명
완전미분방정식 적분인자 증명
$P(x,y)\,dx + Q(x,y)\,dy = 0$ 이 완전미분방정식이 아닐 경우, $FP\,dx + FQ\,dy = 0$ 을 완전하게 만드는 적분인자가 아래와 같음을 증명하라. $$F(x) = \exp\!\int \frac{1}{Q}\!\left(\frac{\partial P}{\partial y} - \frac{\partial Q}{\partial x}\right)dx \qquad F^*(y) = \exp\!\int \frac{1}{P}\!\left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)dy$$
📚 관련 이론: 완전미분방정식 & 적분인자
  • 완전미분방정식 조건: $M\,dx + N\,dy = 0$ 이 완전 $\Leftrightarrow \dfrac{\partial M}{\partial y} = \dfrac{\partial N}{\partial x}$. 이를 만족하면 포텐셜 함수 $u(x,y)$ 가 존재해 $du = 0$, 즉 해가 $u = C$.
  • 적분인자의 목적: 비완전 방정식에 함수 $F$를 곱해 완전하게 만드는 인자. $F$가 $x$ 또는 $y$ 만의 함수일 때 닫힌 형태의 공식이 존재.
  • $F(x)$ 유도: $FP\,dx + FQ\,dy = 0$ 이 완전이 되려면 $\dfrac{\partial(FP)}{\partial y} = \dfrac{\partial(FQ)}{\partial x}$. $F$가 $x$만의 함수이므로 $F\dfrac{\partial P}{\partial y} = F'\!Q + F\dfrac{\partial Q}{\partial x}$, 정리하면 $\dfrac{F'}{F} = \dfrac{1}{Q}\!\left(\dfrac{\partial P}{\partial y} - \dfrac{\partial Q}{\partial x}\right)$ (우변이 $x$만의 함수여야 함). 분리변수형 ODE로 적분하면 $F(x) = \exp\!\int\!\dfrac{1}{Q}\!\left(\dfrac{\partial P}{\partial y} - \dfrac{\partial Q}{\partial x}\right)dx$.
  • $F^*(y)$ 유도: 동일하게 $F$를 $y$만의 함수로 가정. $\dfrac{\partial(F^*P)}{\partial y} = \dfrac{\partial(F^*Q)}{\partial x}$ 에서 $\dfrac{(F^*)'}{F^*} = \dfrac{1}{P}\!\left(\dfrac{\partial Q}{\partial x} - \dfrac{\partial P}{\partial y}\right)$ 가 $y$만의 함수일 때 적분하여 유도.
✏️ Summary

완전 조건 $\partial M/\partial y = \partial N/\partial x$을 $F$를 곱한 방정식에 적용하면, $F$가 $x$만의 함수일 때 $F'/F = \tfrac{1}{Q}(\partial P/\partial y - \partial Q/\partial x)$라는 분리변수형 ODE가 유도된다. 우변이 $x$만의 함수인 경우에만 적분인자가 존재하며, 적분하여 $F(x) = \exp\!\int\tfrac{1}{Q}(\partial P/\partial y - \partial Q/\partial x)\,dx$를 얻는다. $F^*(y)$는 $F$를 $y$만의 함수로 가정해 대칭적으로 유도한다.

2
비제차 1계 선형 ODE 일반해 유도
선형 ODE 적분인자법 일반해 유도
비제차 선형상미분방정식의 일반해를 구하는 과정을 설명하고 유도하라. $$y' + p(x)\,y = r(x)$$
📚 관련 이론: 적분인자법(Integrating Factor Method)
  • 제차해: $r(x)=0$이면 분리변수법으로 $y_h = C\,e^{-\int p\,dx}$ 획득.
  • 적분인자 설정: $h(x) = e^{\int p(x)\,dx}$. 이를 양변에 곱하면 좌변이 완전 미분으로 변환됨.
  • 핵심 변환: $h(x)\,y' + h(x)\,p(x)\,y = \bigl(h(x)\,y\bigr)'$ (곱의 미분 규칙)
  • 적분: $(h\,y)' = h\,r(x)$ 양변을 $x$로 적분 → $h\,y = \int h\,r(x)\,dx + C$
  • 일반해: $$y = e^{-\int p\,dx}\!\left[\int e^{\int p\,dx}\,r(x)\,dx + C\right]$$
  • 구조: 일반해 = 제차해($C$ 항) + 특수해(적분 항)의 합으로 구성됨.
✏️ Summary

$y' + p(x)y = r(x)$에 적분인자 $h = e^{\int p\,dx}$를 곱하면 좌변이 $(hy)'$로 묶힌다. 양변을 적분하면 $hy = \int hr\,dx + C$이므로 일반해는 $y = e^{-\int p\,dx}\!\bigl[\int e^{\int p\,dx}\,r(x)\,dx + C\bigr]$. 제차해와 특수해의 중첩(superposition)으로 이루어진다.

3
베르누이 방정식 / 동차형 ODE
베르누이 방정식 동차형 ODE 선형화
$$x^3 y' = x^2 y + y^3$$
📚 관련 이론
  • 표준화: $x^3$으로 나누면 $y' = \dfrac{y}{x} + \dfrac{y^3}{x^3}$, 즉 $y' - \dfrac{y}{x} = \dfrac{y^3}{x^3}$ (베르누이, $n = 3$)
  • 베르누이 방정식: $y' + p(x)\,y = q(x)\,y^n$에서 $v = y^{1-n}$으로 치환하면 선형 ODE로 변환.
  • 치환: $v = y^{-2}$, $v' = -2y^{-3}y'$. 방정식 양변을 $y^3$으로 나누면 $y^{-3}y' - y^{-2}/x = 1/x^3$.
  • 선형화: $-v'/2 - v/x = 1/x^3$ → $v' + \dfrac{2}{x}v = -\dfrac{2}{x^3}$
  • 적분인자: $x^2$을 곱하면 $(x^2 v)' = -2/x$ → 적분: $x^2 v = -2\ln|x| + C$
  • 동차형 관점: $v = y/x$ 치환 시 $xv' = v^3$ (분리변수형)으로도 풀 수 있으며 동일한 해.
✏️ Summary

$x^3$으로 나누어 베르누이형($n=3$)으로 정리한 뒤, $v = y^{-2}$ 치환으로 선형 ODE $v' + 2v/x = -2/x^3$를 유도. 적분인자 $x^2$으로 풀면 $x^2 v = -2\ln|x| + C$이므로 일반해 $y^2 = \dfrac{x^2}{C - 2\ln|x|}$.

4
치환법 — $v = y + ax$ 형태
치환법 분리변수
$$y' = (y + 4x)^2$$
📚 관련 이론
  • 치환 패턴: $y' = f(ax + by + c)$ 형태는 $v = ax + by + c$ 로 치환하면 분리변수형 ODE로 변환된다.
  • 치환: $v = y + 4x$ → $\dfrac{dv}{dx} = y' + 4 = v^2 + 4$
  • 분리변수: $\dfrac{dv}{v^2 + 4} = dx$
  • 적분: $\displaystyle\int \frac{dv}{v^2 + 4} = \frac{1}{2}\arctan\frac{v}{2}$이므로 $\dfrac{1}{2}\arctan\dfrac{v}{2} = x + C$
  • 역치환: $v = 2\tan(2x + C_1)$, $y = v - 4x$
✏️ Summary

$v = y + 4x$ 치환으로 $v' = v^2 + 4$ (분리변수형)를 얻는다. 적분하면 $\tfrac{1}{2}\arctan(v/2) = x + C$이므로 역치환 후 일반해 $y = 2\tan(2x + C) - 4x$.

5
적분인자로 완전미분방정식화
완전미분방정식 적분인자 $F^*(y)$ 포텐셜 함수
$$(e^{x+y} + ye^y)\,dx + (xe^y - 1)\,dy = 0$$
📚 관련 이론
  • 완전성 검사: $P = e^{x+y} + ye^y = e^y(e^x + y)$, $Q = xe^y - 1$
  • $\dfrac{\partial P}{\partial y} = e^{x+y} + e^y + ye^y,\quad \dfrac{\partial Q}{\partial x} = e^y$ → 불일치, 비완전
  • 적분인자 탐색: $$\frac{\partial Q/\partial x - \partial P/\partial y}{P} = \frac{-e^{x+y} - ye^y}{e^y(e^x+y)} = -1 \quad (y\text{ 만의 함수!})$$
  • 따라서 $F^*(y) = e^{\int(-1)\,dy} = e^{-y}$
  • $e^{-y}$ 곱 후: $P^* = e^x + y$, $Q^* = x - e^{-y}$
  • $\partial P^*/\partial y = 1 = \partial Q^*/\partial x$ → 완전!
  • 포텐셜 함수: $\partial u/\partial x = e^x + y$ → $u = e^x + xy + g(y)$, $\partial u/\partial y = x + g'(y) = x - e^{-y}$ → $g(y) = e^{-y}$
✏️ Summary

$(\partial Q/\partial x - \partial P/\partial y)/P = -1$ (상수, $y$만의 함수)이므로 적분인자 $F^*(y) = e^{-y}$. 곱하면 완전미분방정식이 되고, 포텐셜 함수를 구하면 일반해 $e^x + xy + e^{-y} = C$.

6
분리변수법
분리변수법
$$dy - y^2\sin x\,dx = 0$$
📚 관련 이론: 분리변수법(Separation of Variables)
  • 분리: $dy/y^2 = \sin x\,dx$ → 좌변은 $y$ 함수, 우변은 $x$ 함수로 분리
  • 적분: $\displaystyle\int y^{-2}\,dy = -\frac{1}{y}$, $\displaystyle\int \sin x\,dx = -\cos x$
  • 결합: $-1/y = -\cos x + C$
  • 정리: $y = \dfrac{1}{\cos x + C}$ (단, $y = 0$은 자명해이지만 $y \neq 0$ 범위에서 유효)
✏️ Summary

변수를 분리하여 $y^{-2}\,dy = \sin x\,dx$. 양변 적분 후 정리하면 일반해 $y = \dfrac{1}{\cos x + C}$.

완전미분방정식 — 직접 인식
완전미분방정식 포텐셜 함수
$$(3ye^{3x} - 2x) + e^{3x}y' = 0$$
📚 관련 이론
  • 표준형: $P\,dx + Q\,dy = 0$으로 쓰면 $P = 3ye^{3x} - 2x$, $Q = e^{3x}$
  • 완전성 검사: $\dfrac{\partial P}{\partial y} = 3e^{3x} = \dfrac{\partial Q}{\partial x}$ → 완전!
  • 빠른 인식: $e^{3x}\,dy + 3ye^{3x}\,dx = d(ye^{3x})$이고 $-2x\,dx = d(-x^2)$이므로 직접 $d(ye^{3x} - x^2) = 0$ 인식 가능
  • 포텐셜 함수: $\partial u/\partial y = e^{3x}$ → $u = ye^{3x} + g(x)$, $\partial u/\partial x = 3ye^{3x} + g'(x) = P$ → $g'(x) = -2x$ → $g(x) = -x^2$
  • 해: $u = ye^{3x} - x^2 = C$
✏️ Summary

$\partial P/\partial y = \partial Q/\partial x = 3e^{3x}$으로 완전미분방정식. $e^{3x}dy + 3ye^{3x}dx = d(ye^{3x})$를 직접 인식하거나 포텐셜 함수를 구해 일반해 $ye^{3x} - x^2 = C$, 즉 $y = (C + x^2)e^{-3x}$.

1계 선형 ODE (비제차) — 공명형
선형 ODE 적분인자법 공명
$$y' + ky = e^{-kx}$$
📚 관련 이론
  • 표준형 확인: $p(x) = k$ (상수), $r(x) = e^{-kx}$
  • 적분인자: $h = e^{\int k\,dx} = e^{kx}$
  • 변환: $(e^{kx}y)' = e^{kx} \cdot e^{-kx} = 1$
  • 적분: $e^{kx}y = x + C$
  • 일반해: $y = (x + C)e^{-kx}$
  • 공명 현상: $r(x) = e^{-kx}$가 제차해 $y_h = Ce^{-kx}$와 동일한 지수를 가지므로, 특수해에 $x$ 인수가 등장한다. (2계 ODE의 공명과 유사한 구조)
✏️ Summary

적분인자 $e^{kx}$를 곱하면 $(e^{kx}y)' = 1$. 적분 후 일반해 $y = (x + C)e^{-kx}$. $r(x)$가 제차해와 같은 꼴이기 때문에 특수해에 인수 $x$가 나타나는 공명(resonance) 구조이다.

베르누이 방정식 / 분리변수 — 로지스틱형
베르누이 방정식 분리변수 평형해
$$y' = y(1 - 2y)$$
📚 관련 이론
  • 로지스틱형: $y' = ay - by^2$ 구조. 평형해 $y' = 0$ → $y = 0$ (불안정), $y = 1/2$ (안정 평형)
  • 베르누이 접근 ($n=2$): $y' - y = -2y^2$. $v = y^{-1}$ 치환, 양변을 $y^2$으로 나누면 $y^{-2}y' - y^{-1} = -2$, 즉 $-v' - v = -2$ → $v' + v = 2$
  • 선형 ODE 풀이: 적분인자 $e^x$ 적용 → $(e^x v)' = 2e^x$ → $v = 2 + Ce^{-x}$
  • 역치환: $1/y = 2 + Ce^{-x}$ → $y = \dfrac{1}{2 + Ce^{-x}}$
  • 분리변수 접근: $\dfrac{dy}{y(1-2y)} = dx$, 부분분수 $\dfrac{1}{y(1-2y)} = \dfrac{1}{y} + \dfrac{2}{1-2y}$로 적분해도 동일한 해.
✏️ Summary

베르누이($n=2$), $v = y^{-1}$ 치환 → 선형 ODE $v' + v = 2$. 적분인자 $e^x$로 풀면 $v = 2 + Ce^{-x}$. 일반해 $y = \dfrac{1}{2 + Ce^{-x}}$. 평형해: $y=0$ (불안정), $y=1/2$ (안정).

베르누이 방정식 — 초기값 문제(IVP)
베르누이 방정식 초기값 문제 부분적분
$$xy' + y = y^2 \ln x, \qquad y(1) = \tfrac{1}{2}$$
📚 관련 이론
  • 표준화: $x$로 나누면 $y' + \dfrac{1}{x}y = \dfrac{\ln x}{x}\,y^2$ → 베르누이 ($n=2$)
  • 치환: $v = y^{-1}$, $v' = -y^{-2}y'$. $y^2$으로 나누면: $y^{-2}y' + \dfrac{y^{-1}}{x} = \dfrac{\ln x}{x}$
  • 선형화: $-v' + \dfrac{v}{x} = \dfrac{\ln x}{x}$ → $v' - \dfrac{v}{x} = -\dfrac{\ln x}{x}$
  • 적분인자: $e^{-\int(1/x)\,dx} = e^{-\ln x} = 1/x$. $(v/x)' = -\dfrac{\ln x}{x^2}$
  • 적분 (부분적분 필요): $\displaystyle\int \frac{\ln x}{x^2}\,dx = -\frac{\ln x}{x} - \frac{1}{x} + C$
  • 따라서 $v/x = \dfrac{\ln x}{x} + \dfrac{1}{x} + C$ → $v = \ln x + 1 + Cx$
  • 초기조건: $y(1) = 1/2$ → $v(1) = 2$: $2 = 0 + 1 + C \cdot 1$ → $C = 1$
✏️ Summary

베르누이($n=2$), $v = y^{-1}$ 치환 → 선형 ODE $v' - v/x = -(\ln x)/x$. 적분인자 $1/x$와 부분적분으로 $v = \ln x + 1 + Cx$. 초기조건 $y(1) = 1/2$에서 $C = 1$이므로 특수해 $y = \dfrac{1}{\ln x + x + 1}$.

로지스틱 모형 — 개체군 동역학
로지스틱 방정식 분리변수 / 부분분수 평형해 & 안정성 초기값 3가지
아래 로지스틱 모형에서 초기조건 $y(0) = 10,\;100,\;1000$ 일 때 각각의 해를 구하고 그래프를 그려 의미를 설명하라. $$y' = 100y - y^2 = y(100 - y)$$
📚 관련 이론: 로지스틱 방정식(Logistic Equation)
  • 로지스틱 방정식 표준형: $y' = ky\!\left(1 - y/M\right)$. 여기서 $k = 100$, 환경 수용력(carrying capacity) $M = 100$
  • 평형해(Equilibrium): $y' = 0$ → $y = 0$ (불안정), $y = 100$ (안정). $0 < y < 100$이면 $y$ 증가, $y > 100$이면 $y$ 감소.
  • 분리변수법: $\dfrac{dy}{y(100-y)} = dx$
  • 부분분수 분해: $\dfrac{1}{y(100-y)} = \dfrac{1}{100}\!\left(\dfrac{1}{y} + \dfrac{1}{100-y}\right)$
  • 적분: $\dfrac{1}{100}\ln\!\left|\dfrac{y}{100-y}\right| = x + C$ → 지수화 → $\dfrac{y}{100-y} = Ae^{100x}$
  • 일반해: $y = \dfrac{100}{1 + Be^{-100x}}$, 여기서 $B = \dfrac{100 - y_0}{y_0}$
  • 초기조건별 해:
    • $y_0 = 10$: $B = 9$ → $y = \dfrac{100}{1 + 9e^{-100x}}$ (S자형 증가, 서서히 100으로 수렴)
    • $y_0 = 100$: $B = 0$ → $y = 100$ (안정 평형, 변화 없음)
    • $y_0 = 1000$: $B = -\tfrac{9}{10}$ → $y = \dfrac{100}{1 - 0.9\,e^{-100x}}$ (단조 감소, 100으로 수렴)
  • 그래프 해석: 초기값에 관계없이 $x \to \infty$이면 $y \to 100$. 개체군이 환경 수용력에 의해 제한됨을 의미.
✏️ Summary

$y' = y(100-y)$는 수용력 $M=100$인 로지스틱 방정식. 분리변수 후 부분분수 적분으로 일반해 $y = 100/(1 + Be^{-100x})$, $B = (100-y_0)/y_0$. $y_0 = 10$이면 S자형 증가, $y_0 = 100$이면 평형 유지, $y_0 = 1000$이면 단조 감소하며, 세 경우 모두 안정 평형 $y = 100$으로 수렴한다. 이는 개체군이 아무리 많거나 적더라도 결국 환경 수용력으로 수렴함을 나타낸다.
